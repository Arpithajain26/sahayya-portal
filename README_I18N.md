# i18n Integration Notes

This project has been updated to include internationalization support using i18next.

Steps to run locally after pulling the changes:

```bash
cd sahayya-portal
npm install
# install i18n packages
npm install i18next react-i18next
npm run dev
```

The Language selector is available on the Index page. Translation files live in `src/locales/{en,hi,te,kn}/translation.json`.
